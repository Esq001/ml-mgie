# Refined III for Erin

Three Lightroom presets recreated in the spirit of the Refined Co. **Refined III** pack
(a Fuji 400H / Portra 400-style film emulation). These are clean-room recreations
based on publicly described characteristics of the pack — they aren't the originals
and aren't affiliated with Refined Co.

![Mockup: original vs. Refined III.1 / III.2 / III.3 across five sample photos](MOCKUP.png)

> The mockup above is rendered by an in-repo simulator (`backtest/`) — directionally
> accurate but not pixel-identical to Adobe Camera Raw. Final visual approval
> happens in Lightroom.

## What's in the pack

| Preset            | Look                                                                                              |
| ----------------- | ------------------------------------------------------------------------------------------------- |
| **Refined III.1** | Deep filmy cyan greens, clean luminous **peach** skin tones with delicate pinks.                  |
| **Refined III.2** | Deep filmy cyan greens, delicate cyan shadows, **warm luminous** skin tones and highlights.      |
| **Refined III.3** | Warmer cyan greens, cyan shadows, warm skin tones with a splash of **cool pink** in highlights.   |

All three lean on lifted blacks, soft highlight rolloff, muted yellows, cyan-shifted
greens, and gentle film grain — the classic 400H signature.

## Install — Lightroom Classic (desktop)

1. In Lightroom Classic, go to the **Develop** module.
2. Open the **Presets** panel (left side), click the **+** icon → **Import Presets…**.
3. Select all three `.xmp` files in this folder.
4. They'll appear under a group named **Refined III for Erin**.

## Install — Lightroom CC / Mobile (cloud / phone)

1. On desktop Lightroom (cloud version), click the **Presets** panel → **…** → **Import Presets**.
2. Pick the three `.xmp` files.
3. Once synced, they'll appear on Erin's iPad and phone in the same **Refined III for Erin** group.

> Lightroom Mobile alone can also import `.xmp` directly via **Presets → ⋯ → Import Presets**
> if you AirDrop / email the files to the device.

## Usage tips

- Designed for **portrait, wedding, and lifestyle** work in good natural light.
- After applying, nudge **white balance** first — film looks live or die on WB.
- Each preset supports the **Amount** slider (Lightroom 12.3+) so Erin can dial the
  intensity up or down without rebuilding settings.
- Grain is set subtle (12–14). Bump to 18–22 for a stronger film feel on smooth skin.
- If skin runs too peach on very fair complexions, drop **Orange Saturation** another -5.
